# 🤖 AI智能助手 - Web版

一个功能完整、界面现代的Web应用程序，集成了大模型对话与文件分析功能。本项目是我为了重新掌握全栈开发技能而构建的实践作品，从零到一实现了从后端API调用到前端界面交互的完整流程。

**在线演示**：`https://你的项目名.vercel.app` （部署在Vercel）

## ✨ 功能特性

- **💬 智能对话**：与先进的大模型进行自然、多轮对话。
- **📁 文件分析**：上传文本文件（.txt, .md, .pdf），AI可基于文件内容进行总结、问答和分析。
- **🌐 现代Web界面**：采用Bootstrap 5构建的响应式界面，美观且跨设备兼容。
- **🚀 一键部署**：配置完善，可快速部署至Vercel等云平台。

## 🛠️ 技术栈

| 层面 | 技术选型 |
|------|----------|
| **后端框架** | Python, Flask |
| **AI能力** | 智谱AI GLM大模型 API |
| **前端界面** | HTML5, Bootstrap 5, Jinja2模板 |
| **部署平台** | Vercel (Serverless) |
| **版本控制** | Git, GitHub |
