关于此项目

这是一个为重新掌握计算机专业技能而进行的学习与实践项目。

作为一名有近三年空窗期的计算机专业毕业生，我通过这个项目系统地重拾了从Python基础、API调用、到全栈Web开发（Flask， HTML/CSS， Bootstrap）等一系列核心技能。

🎯 项目目标
  • 构建一个真正可用、功能完整的AI应用，作为技术能力的证明。
  • 实践从项目规划、编码、调试到部署的完整开发流程。
  • 理解并应用现代Web开发的最佳实践与项目结构。
🛠️ 技术栈
  • 后端: Python, Flask, 智谱AI/OpenAI API
  • 前端: HTML5, Bootstrap 5, Jinja2 模板
  • 部署: (计划部署到) Vercel / Railway
  • 开发工具: Git, GitHub, VSCode
